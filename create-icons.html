🎨 MISSKA — Generator Icon PWA

Buat semua icon yang dibutuhkan untuk PWA MISSKA. Klik "Generate & Download Semua" lalu pindahkan file ke folder icons/.

✅ Semua icon berhasil dibuat! Pindahkan file ke folder icons/ di direktori yang sama dengan misska.html

📋 Cara Menggunakan

  1. Klik tombol "Generate & Download Semua Icon"
  2. Browser akan mendownload 8 file PNG (icon-72.png sampai icon-512.png)
  3. Pindahkan semua file tersebut ke folder icons/ di direktori yang sama dengan misska.html
  4. Buka misska.html melalui web server (bukan file://)
  5. Di Chrome, klik ikon install di address bar atau tunggu banner muncul

💡 Cara buka via web server lokal:
  1. Install VS Code Extension: Live Server
  2. Klik kanan misska.html → Open with Live Server
  3. Aplikasi terbuka di http://127.0.0.1:5500/misska.html
  4. Di HP, buka browser Chrome → ketik IP komputer: http://192.168.x.x:5500/misska.html
